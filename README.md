# weather-api

A weather app made with vanilla async javascript

# API data retrieved

## Current:

Current Object:
-condition: [text: string, icon: imgurl],
-humidity: num,
-gust_kph: num,
-is_day: num,
-last_updated: date,
-precip_in: num,
-precip_mm: num
-temp_c: num,
-temp_f: num,
-wind_kp: num

Location Object:
-localtime: date,
-name: string,
-region: string
